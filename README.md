# Dermacare

A dermatology care application with user profile management.

## Features

- **User Profile Page**: A responsive user profile interface with the following features:
  - View and edit personal information (name, email, phone, date of birth, address)
  - Upload and display profile picture
  - Manage medical history
  - Responsive design that works on desktop and mobile devices
  - Smooth animations and modern UI

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Click "Edit Profile" to modify your information
4. Click "Save Changes" to save or "Cancel" to discard changes

## Files

- `index.html` - Main user profile page
- `styles.css` - Styling for the profile page
- `script.js` - Interactive functionality for editing and saving profile data

## Technology Stack

- HTML5
- CSS3 (with responsive design)
- Vanilla JavaScript
